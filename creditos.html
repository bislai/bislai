Todos los datos de las mociones han sido obtenidos de la web del Ayuntamiento de Zaragoza. En un futuro no muy lejano la idea es presentar los datos de las mociones como objetos JSON siguiendo los patrones establecidos en la API del Ayuntamiento de Zaragoza.

Esta web no hubiera sido posible sin la generosa ayuda de librerías open source.

Para las gráfica animadadas he utilizado mo.js de Oleg Solomka.

Para las gráficas de las mociones he utilizado c3.js de Masayuki Tanaka.

Para las gráficas de las estadisticas he utilizado chart.js de Nick Downie, y que ahora es mantenida por la comunidad.

Para la iconografía he utilizado iconos de materialdesing.io.
