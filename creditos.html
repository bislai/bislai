ESTADÍSTICAS PARTIDOS DATOS

Todos los datos de las mociones han sido obtenidos de la web del Ayuntamiento de Zaragoza.

Esta web no hubiera sido posible sin la generosa ayuda de librerías e iconos distribuídos como open source.

Librerías

  • Para las gráficas de las mociones he utilizado c3.js de Masayuki Tanaka.
  • Para las gráficas de las estadísticas he utilizado chart.js de Nick Downie, y que ahora es mantenida por la comunidad.
  • Para la arquitectura CSS he utilizado Astral, un framework CSS hecho a mi medida y que esta fuermente inspirado en Tachyons y Basscss.
  • Para el buscador he utilizado Tipue-Search que esta desarrollado por Tipue.

Iconos

  • Para la iconografía he utilizado iconos de Material.

  • El icono que acompaña las gráficas de votos a favor ha sido diseñador por Made Somewhere on the @NounProject.
